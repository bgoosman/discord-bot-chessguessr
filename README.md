A discord bot which records chessguessr results to chessguessr.db.

```
Chessguessr #713 3/5

游릴拘럭샢拘럭샢
游릴游릴拘럭샢游쥟릖
游릴游릴游릴游릴游릴

https://chessguessr.com/

# Will output
> {"user": "user1", "games_played": 1, "win_percent": 1.0}
```

# Commands

`$leaderboard`: prints a leaderboard. e.g.

```
{"user1": 1, "user2": 1}
```
